# VisualTimeSeries

## Get started

### 1.Add your API keys 

Before you get started, it is necessary to get API keys of LLMs. In our framework, you should create a .yaml file called `api_keys.yaml` in `BigModel/` directory. The format is shown below:

```yaml
openai:
  api_key: 'Your API Keys'
chatglm:
  api_key: 'Your API Keys'
```
